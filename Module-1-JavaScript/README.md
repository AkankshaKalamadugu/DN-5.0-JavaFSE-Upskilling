# Module 1 - JavaScript

## Local Community Event Portal

This project was created to practice **JavaScript concepts** using the theme **Local Community Event Portal**. The portal allows users to view community events, register for events, submit feedback, save preferences, and interact with the webpage dynamically using JavaScript.

The main goal of this module was to understand how JavaScript works with HTML and CSS to make a webpage interactive.

---

## Task 1: JavaScript Basics and Setup

* Linked JavaScript file using `main.js`
* Used `console.log()` to display a welcome message
* Added an alert message when the page loads

---

## Task 2: Syntax, Data Types and Operators

* Used `const` for event name and date
* Used `let` for available seats
* Displayed event details using template literals
* Used seat count decrement (`--`) for registration
