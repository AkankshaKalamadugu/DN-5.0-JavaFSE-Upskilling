# Bootstrap 5 

## Project Overview

The Local Community Event Portal** is a responsive web page developed using **HTML5, Bootstrap 5, and Bootstrap Icons**. This project is created to practice Bootstrap concepts and understand how Bootstrap components can be used to build a simple and attractive website.

The portal is designed for a local community where users can view upcoming events, explore event details, register for activities, and read frequently asked questions. The webpage is responsive, so it works properly on different screen sizes such as mobile phones, tablets, and desktops.

## Objective of the Project

The main objective of this project is to practice Bootstrap 5 and learn how to design a responsive webpage using ready-made Bootstrap components and utility classes.

This project helps in understanding:

- Responsive webpage design
- Bootstrap grid system
- Navbar creation
- Cards and forms
- Buttons and alerts
- Modal popup
- Accordion section
- Utility classes for spacing, colors, borders, shadows, and positioning

## Features Implemented

### 1. Responsive Navigation Bar

A responsive navigation bar is used at the top of the webpage. It helps users move to different sections of the portal easily. The navbar adjusts according to the screen size.

### 2. Bootstrap Grid Layout

The Bootstrap grid system is used to arrange content properly. Event cards are displayed in a responsive layout so that they look good on desktop, tablet, and mobile screens.

### 3. Community Event Cards

Event cards are used to display different community events. Each card contains event information in a clean and organized format.

### 4. Typography Utilities

Bootstrap typography classes are used to style headings, paragraphs, and text content. This improves the readability of the webpage.

### 5. Registration Form

A registration form is included for users to register for community events. The form contains input fields and buttons styled using Bootstrap form classes.

### 6. Buttons and Button Styles

Different Bootstrap button styles are used in the webpage. Buttons are used for actions like registration, viewing details, and opening modal popups.

### 7. Alerts and Badges

Bootstrap alerts and badges are added to highlight important information such as event status, notifications, or special labels.

### 8. Modal Popup

A Bootstrap modal popup is used to display additional information without opening a new page.

### 9. Accordion FAQ Section

An accordion section is included for frequently asked questions. It allows users to expand and collapse answers easily.

### 10. Colors and Background Utilities

Bootstrap color and background classes are used to improve the appearance of the webpage.

### 11. Display and Visibility Utilities

Bootstrap display classes are used to control how elements appear on different screen sizes.

### 12. Borders, Shadows, and Rounded Corners

Bootstrap border, shadow, and rounded classes are used to give the webpage a clean and modern look.

### 13. Positioning Utilities

Positioning utilities are used to place elements properly on the page.

### 14. Footer with Social Media Icons

A footer section is added at the bottom of the webpage. It includes social media icons using Bootstrap Icons.
