## Local Community Event Portal

This project was created to practice **CSS3 concepts** by designing a **Local Community Event Portal** website. Different CSS properties were used to improve the design, alignment, colors, typography, tables, responsive layout, and overall user experience.

---

## Task 1: CSS Introduction

In this task, different ways of applying CSS were explored.

Implemented:
- Inline CSS for simple styling
- Internal CSS using the `<style>` tag
- External CSS using `styles.css`
- CSS comments for better understanding

---

## Task 2: CSS Syntax and Comments

Implemented proper CSS syntax and formatting.

Used:
- Selectors
- Properties and values
- Comments for code readability
- Proper indentation

Example:

```css
button {
    background-color: green;
    color: white;
}
```

---

## Task 3: CSS Selectors

Different CSS selectors were used in the project.

Implemented:
- Universal Selector (`*`)
- Element Selector (`h2`)
- ID Selector (`#mainHeader`)
- Class Selector (`.eventCard`)
- Grouping Selector (`h3, p`)

These selectors were used to style headings, event cards, and text sections.

---

## Task 4: Colors and Background Styling

Applied different background and color properties.

Implemented:
- HEX colors
- RGBA colors
- Background image
- Background size and positioning
- Background attachment

Used a background image to improve the visual appearance of the portal.

---

## Task 5: Typography and Fonts

Improved the website text styling using typography properties.

Implemented:
- Google Fonts (**Poppins**)
- `font-family`
- `font-size`
- `font-weight`
- `line-height`
- `letter-spacing`
- `text-transform`

These styles improved readability and appearance.

---

## Task 6: Link and Navigation Styling

Styled navigation links and menu items.

Implemented:
- `:link`
- `:visited`
- `:hover`
- `:active`

Also added spacing and styling to navigation links for better UI.

---

## Task 7: Table Styling

Styled the event gallery table.

Implemented:
- Table borders
- Padding and spacing
- `border-collapse`
- `text-align: center`
- Zebra striping using:

```css
tr:nth-child(even)
```

This improved the appearance of the community event gallery.

---

## Task 8: Box Model and Layout

Used CSS box model properties for proper alignment.

Implemented:
- `margin`
- `padding`
- `border`
- `outline`

Applied styling to event cards and form fields.

Also practiced:
- `visibility: hidden`
- `display: none`

---

## Task 9: Multi-Column Layout

Created a **Community News Bulletin** section.

Implemented:
- `column-count`
- `column-gap`
- `column-rule`

This created a newspaper-style text layout.

---

## Task 10: Responsive Design

Implemented responsive web design using media queries.

Features:
- Mobile-friendly layout
- Responsive images
- Responsive text sizes
- Navigation changes for smaller screens

Used:

```css
@media screen and (max-width: 768px)
```

to adjust layout for mobile devices.

---

## Task 11: Testing and Debugging

Tested the website using **Chrome DevTools**.

Performed:
- Responsive testing
- Mobile screen simulation
- Style inspection
- Layout debugging

Verified that the website works properly on different screen sizes.
